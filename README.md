# string-utils-demo

Petites fonctions utilitaires de manipulation de chaînes de caractères, en Python.

## Fonctions

- `is_palindrome(text)` — vérifie si un texte est un palindrome (ignore casse et ponctuation).
- `reverse_words(text)` — inverse l'ordre des mots dans une phrase.
- `count_vowels(text)` — compte le nombre de voyelles dans un texte.
